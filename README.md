# なんでもホッチキス
## About
PDF、画像、Wordファイルなど、多種多様なファイル形式を雑多にぶち込んで繋げることが出来るWebサービスです。

## 動作環境
### 依存ライブラリ
- Flask
- PyMySQL
- bcrypt
- img2pdf
- PyPDF2
### 依存ソフトウェア
- LibreOffice
